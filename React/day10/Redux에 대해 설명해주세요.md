### 동우

- redux는 자바스크립트의 상태관리 라이브러리이며, 자바스크립트 앱을 위한 예측 가능한 상태 컨테이너이다.
- 핵심 키워드로는 Store, Action, Type, Reducer, dispatch 등이 있습니다.
- **Store**: 상태가 관리되는 **오직 하나의 공간** 이다. 앱에서 필요한 상태들과 리듀서가 저장되어 있으며, 컴포넌트에서 상태 정보가 필요할 때 스토어에 접근한다.
- **Action**: 상태를 변화시키려는 의도를 표현한 객체이다. 상태를 변경해야 할 때, 어떠한 변화를 줄지에 해당하는 액션을 발생시킬 수 있다.
  - action type과 payload로 이루어짐
  - Action Creator: 동작에 대해 선언된 Action을 생성해 실제 객체로 만들어주는 함수이다.
- Reducer: Action과 이전 state를 인자로 받아 실제 state의 변화를 일으키는 함수이다.
  => store에 접근해 action에 맞춰 state를 변경한다. reducer는 순수함수임을 지켜야 하는 원칙으로, 이전의 상태는 건드리지 않고 변화된 새로운 상태 객체를 만들어 반환한다.
- Dispatch: store의 내장 함수 중 하나로, action을 파라미터로 전달받고 reducer를 실행시킨다.
- Subscribe: store의 내장 함수 중 하나로 ,특정 함수를 전달해주면 action이 dispatch 되었을 때마다 전달된 함수가 전달된다.

### 은지

- redux는 상태 관리 라이브러리입니다. flux 아키텍처의 구현체로 대형 MVC 애플리케이션에서 종종 나타나는 데이터 간 의존성 이슈로 인해 데이터의 흐름을 예측할 수 없었던 문제를 해결하기 위해 고안되었습니다.
- 사례: 페이스북의 채팅 버그, 읽지 않은 메시지를 사용자가 확인해도 메시지 카운터가 사라지지 않는 문제를 발견했고 페이스북 팀은 설계에 기반한 근본적인 문제를 해결하기 위해 단방향 데이터 흐름을 활용해 예측 가능한 시스템을 만들게 되었다.

<br />

### 지연

- ‘액션’이라는 이벤트를 사용해 어플리케이션 상태를 관리하고 업데이트하기 위한 패턴 및 자바스크립트 라이브러리
- 상태가 예측 가능한 방식으로만 업데이트될 수 있도록 하는 규칙과 함께 **전체** 어플리케이션에서 사용해야 하는 상태에 대한 중앙 집중식 저장소 역할

<br />

### 규란

- 리덕스는 자바스크립트 상태관리 라이브러리입니다. react, vue, vanilla js 등 뷰 라이브러리에 구애받지 않고 사용할 수 있습니다.

### 꼬리 질문 모음

- **꼬리 질문 1 - react-redux와 redux의 차이는?**
  - 리덕스는 리액트만을 위한게 아니라 리액트환경에서 리덕스를 사용할 수 있도록 도움을 주는 도구가 있는데 이게 바로 react-redux이다.
- **꼬리 질문 2 - 루트 리듀서를 쓰는 이유는 뭔가요?**

  - Store를 만들 때에는 하나의 리듀서만 사용해야하기 때문이다. 다수의 리듀서들의 combineReducer함수를 통해 하나로 합쳐서 처리한다.

- **꼬리 질문 3 - flux 패턴이 등장한 배경에 대해 설명해주세요.**
  - 대규모 애플리케이션에서 데이터 흐름을 일관성 있게 관리함으로써 프로그램의 예측가능성(Predictability)을 높이기 위함입니다.
  - 기존 대부분의 애플리케이션은 MVC 패턴을 사용했었는데, 이 때 데이터가 양방향으로 흐를 수 있습니다. 모델에서 여러 뷰들을 업데이트하고, 또 그 뷰가 다른 모델을 업데이트하는 식의 복잡한 데이터 흐름을 가지게 됩니다.
  - 이를 해결하기 위해 단방향 데이터 흐름을 가지는 flux 패턴을 고안하게 되었습니다.
- **꼬리 질문 4 - 리듀서란 무엇인가요?**
  - 리듀싱 함수라고 부르기도 하며, 리덕스에서 리듀스란 이전 상태와 액션을 받아 다음 상태를 반환하는 순수함수이다.
  - 함수형 프로그래밍에서 나온 개념으로, 누적값과 값을 받아서 새로운 누적값을 반환하는 함수이다.
  - Redux에서 누적값은 상태 객체이고, 누적될 값은 액션이다. 주어진 이전 상태와 액션에서 새로운 상태를 계산한다.
  - 부수효과를 가져서는 안된다.
